QUIZLY — QUIZ IMPORT TEMPLATES
================================

How to use these files:
1. Go to Dashboard → "Import from text"
2. Enter a title and time limit
3. Copy-paste the contents of any file below into the text area
4. Click "Continue to editor" to preview and adjust
5. Save to publish the quiz

Available templates:
--------------------
anime_general.txt       — 12 questions, mixed anime (Naruto, DBZ, AOT, etc.)
jjk_season1.txt         — 15 questions, Jujutsu Kaisen Season 1
world_geography.txt     — 12 questions, countries, capitals, rivers
world_history.txt       — 12 questions, key events across centuries
science_basics.txt      — 12 questions, physics, biology, chemistry
web_development.txt     — 12 questions, HTML/CSS/JS/HTTP (includes multi-select)

Format reference:
-----------------
Question: Your question text here
A) First option
B) Second option (correct)
C) Third option
D) Fourth option

Rules:
- Each question starts with "Question:"
- Options use A) B) C) D) E) F) format (2–6 options)
- Mark correct answers with (correct) at the end of the line
- For multiple-answer questions, add "(select all that apply)" after the question text
- Max 20 questions per quiz
