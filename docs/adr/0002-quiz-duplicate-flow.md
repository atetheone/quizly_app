# Quiz duplicate copies everything and redirects to edit immediately

Duplicating a Quiz auto-generates a title ("Copy of {original title}"), copies all questions, answer options, and the time limit. No name prompt — the teacher renames on the edit page. After duplication, the teacher is redirected to the new quiz's edit page immediately.

The duplicate button lives on the dashboard card: always available as a secondary action, and the primary action on locked quizzes (where Edit is hidden).
